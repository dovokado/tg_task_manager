# Почни звідси 👋

Цей набір налаштує твій Claude Code так, щоб він працював дисципліновано
й ефективно — і сам по собі, і з твоїми проєктами, і з Obsidian.

**Тобі не треба нічого копіювати вручну.** Усе зробить сам Claude — ти лише
підтверджуватимеш кожен крок. Він пояснюватиме українською, що і навіщо робить.

---

## Що зробити (1 крок)

1. Поклади цю теку `CLAUDE-KIT` будь-куди на свій комп'ютер.
2. Встанови Claude Code, якщо ще не встановлено:
   - постав Node.js LTS з https://nodejs.org
   - у терміналі виконай: `npm install -g @anthropic-ai/claude-code`
   - запусти командою `claude` і увійди в акаунт.
3. Відкрий сесію Claude Code **усередині теки `CLAUDE-KIT`** і встав цей запит:

```
Прочитай файл INSTALL.md у цій теці й виконай усі кроки встановлення.
Спілкуйся зі мною українською. Перед кожною дією, що змінює файли або
налаштування, поясни що і навіщо робиш і дочекайся мого підтвердження.
```

Усе. Далі Claude вестиме тебе сам — твоя справа казати «так» або «ні».

---

## Що буде налаштовано

- **Глобальна поведінка** — Claude питатиме розуміння й план перед діями,
  поважатиме рівні складності, відповідатиме українською.
- **Пам'ять між сесіями** — хто ти, твоя мова, твої правила роботи.
- **Робота з проєктами** — шаблони, синхрон коду й документації, чеклісти.
- **Obsidian + MCP** — Claude зможе працювати з твоїми нотатками напряму.

> Технічні файли набору написані англійською — так Claude найточніше їх виконує.
> Але спілкування з тобою й уся документація твоїх проєктів — українською.
